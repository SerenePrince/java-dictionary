# Contributing

This is a solo project and I'm not accepting pull requests at this time. Issues are open, but I can't promise a quick turnaround.

That said, you're more than welcome to fork the repo and make it your own. If something about the structure, the dual-definition concept, or the general idea is useful to you, take it and run with it. That's what the MIT license is there for.

If you do build something off of this, I'd genuinely love to see it.
